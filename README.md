# 🧠 Master’s Thesis: DEVELOPMENT OF DEEP LEARNING-BASED SENTIMENT ANALYSIS APPROACHES WITH NEURAL NETWORK-BASED LANGUAGE MODELS

## 📌 Overview

This repository presents my Master’s thesis in Computer Engineering, focused on Sentiment Analysis using Deep Learning and Natural Language Processing techniques. The research explores both static and contextualized language models to improve sentiment classification performance on large-scale product review datasets.

---

## 🎯 Research Objective

The main goal of this study is to develop and evaluate advanced sentiment analysis models capable of accurately interpreting user opinions from textual data, especially in large and complex review datasets.

---

## 🧠 Methodology

The research investigates and compares multiple approaches:

### 🔹 Static Language Models (SLMs)

* Word2Vec
* GloVe
* FastText

### 🔹 Contextual Language Models (CLMs)

* BERT
* ELMo

### 🔹 Deep Learning Models

* Convolutional Neural Networks (CNN)
* Long Short-Term Memory (LSTM)
* Bidirectional LSTM (BiLSTM)
* Feed Forward Neural Networks

---

## 📊 Dataset

Experiments were conducted using Amazon product review datasets, which are widely used benchmarks in sentiment analysis research.

---

## 📈 Key Results

The proposed CNN-based BERT model achieved the highest performance, reaching:

* **97% training accuracy**
* **95% testing accuracy**

---

## 💡 Key Contributions

* Comparison between static and contextual embeddings
* Integration of deep learning architectures with language models
* Improved sentiment classification performance on benchmark datasets
* Analysis of model effectiveness in real-world review data
